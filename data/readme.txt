This folder contains the 36-MEG datasets, including (on Jun 1, 2023)

1. individual_connectomes_reordered.nc: The SC for each data
2. individual_psd_reordered_matlab.nc: The PSD for each data
3. meg_individual_tseries_reordered.nc: The time series for each data
4. mean80_fibercount.csv: HCP SC template
5. mean80_fiberlength.csv: HCP dist matrix
6. rois_order.txt: the order of rois. 

The data is in DK atlas (86 ROIs).
